# zlown

Rogue Access Point

## Status

[![Gem Version](https://badge.fury.io/rb/zlown@2x.png)](https://badge.fury.io/rb/zlown)
[![Downloads](http://img.shields.io/gem/dt/zlown.svg)](http://rubygems.org/gems/zlown)

## Features implemented

- Connects to internet using LTE
- Shares connectivity over WiFi
- Logs all HTTP trafic

## Features planned

- HTTPS sniffing
- Evil Twin WiFi
- Auto terminate (RST) connections based on criteria specified (pcap filter)
- On-the-fly trafic modification
- Fake updates

## Prerequisites

- apt-get (developed using [Kali Linux](https://www.kali.org/))
- systemd
- dnsmasq
- hostapd
- ruby
- rubygems
- bundler

## How To

- `gem install zlown`
- `zlown install`
- `zlown init`
- `zlown enable`
- `zlown start`

***OR***

- `git clone https://github.com/korczis/zlown.git`
- `bundle`
- `./bin/zlown install`
- `./bin/zlown init`
- `./bin/zlown enable`
- `./bin/zlown start`
