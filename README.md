# zlown

Rogue Access Point

## Prerequisites

- apt-get (developed using [Kali Linux](https://www.kali.org/))
- systemd
- dnsmasq
- hostapd
- ruby
- rubygems
- bundler

## How To

- `git clone https://github.com/korczis/zlown.git`
- `bundle`
- `./bin/zlown install`
- `./bin/zlown init`
- `./bin/zlown enable`
- `./bin/zlown start`
